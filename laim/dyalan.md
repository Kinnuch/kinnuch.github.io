## 语系概览

```
原祖语 (Proto-Andoric)
├── 古語 (Andoric / 安多语)
│   ├── 嘉兰语 Dyalan（东斯拉夫式演化）
│   └── [其他方言/姊妹语]
└── 中央语（独立体系，仅词汇层面受古語影响）
```

- **原祖语**：词根来源，类PIE地位，含三系软腭、元音交替(ablaut)
- **古語**：宗教经典语言，保留有生性标记(M-X-S结构)
- **嘉兰语**：日常通用语，有生性脱落(M-S结构)，零级形用于复合与派生

## 四、形态结构

### 古語：M-X-S（词根 - 有生性 - 范畴尾缀）

```
┌──────┐  ┌──────────┐  ┌──────────┐
│ 词根 │──│ 有生性标记 │──│ 范畴尾缀 │
│(重音) │  │ (吸引重音) │  │          │
└──────┘  └──────────┘  └──────────┘
  M            X              S
```

### 嘉兰语：M-S（有生性脱落，零级形用于复合/派生）

```
┌────────────┐  ┌──────────┐
│ 词根(零级)  │──│ 范畴尾缀 │
│            │  │          │
└────────────┘  └──────────┘
     M               S
```

**关键规则：**
- 独立词/词典形 → 完整级词根 + 尾缀
- 复合词/派生词 → 零级词根 + 尾缀
- 零级形 = 历史上有生性标记触发的痕迹，现已泛化为构词规则

## 五、有生性系统

### 有生性后缀（古語，中间位置）

| 等级 | 后缀 | 功能 | 嘉兰语中 |
|---|---|---|---|
| 神性 | -ēs- | 标记神级存在，吸引重音 | 完全脱落 |
| 人性 | -on- | 标记人/智慧体，吸引重音 | 完全脱落 |
| 有生 | -ur- | 标记动植物，吸引重音 | 完全脱落 |
| 无生 | ∅ | 无标记 | — |

三种有生性在嘉兰语中合流：只剩"零级/完整级"的二元对立。

### 零级化规则

当有生性后缀出现（或在嘉兰语中进行复合/派生）时，词根元音变化：

| 词根元音 | 零级结果 | 说明 |
|---|---|---|
| 短 \*a, \*e, \*o | → ∅ | 直接消失 |
| 长 \*ē, \*ā, \*ō | → 短 \*e, \*a, \*o | 缩短 |
| 长 \*ī, \*ū | → 短 \*i, \*u → yer | 缩短后变yer |
| 相邻 \*w | → \*u（半元音抬升） | 产生ъ |
| 相邻 \*j | → \*i（半元音抬升） | 产生ь |
| 相邻 \*r, \*l | → 成音节 ьr, ьl | 可能发声为 er, el |
| 相邻 \*m, \*n | → 成音节 ъm, ъn | 可能发声为 om, on |

### 辅助元音规则

- 零级产生的辅音丛若≥4个辅音：插入辅助元音 ь→e
- 词首不可发音丛：词首yer强制发声为e
- 来自原始 \*u 的 ъ 发声为 o；来自零级辅助的 ь 发声为 e
  - 这产生了 **o/e 元音对立**：无标记形含o（来自ъ），零级形含e（来自辅助ь）

## 十一、构词规则总结

1. **独立词** = 完整级词根 + 范畴尾缀
   - Zvalja（光），Tolkja（月），Vīdor（巫师）

2. **复合词** = 零级词根 + 零级词根 + ... + 范畴尾缀
   - Nekr-telkja（血月），Śer-zlja-dor（圣光会）

3. **前缀不触发零级**：ne-（否定），tor-（反），per-（初），nov-（新）
   - Ne-brek Zlja（无垠之光），Tor-anjo（天灾）

4. **数词修饰不触发零级**：
   - Gor-selk Śoljo（双弦密论）— 数词gor不变，被修饰词selk用零级

5. **骂人/讽刺用法**：在嘉兰语中对人使用零级形（暗示"你以为你是神？"）
   - 正常称呼巫师：Vīdor
   - 讽刺：Vdor（"哟，大神巫师啊？"）

## 十二、设计备忘

- 嘉兰人不信神 → 日常用无标记完整级形
- 零级形在嘉兰语中的功能已泛化为"构词/复合"规则，说话者不知其来源是有生性
- 古語经文保留完整的 M-X-S 三段结构，可用于小说中的宗教场景
- 中央语体系独立，暂未设计
- 方言暂不考虑，标准语基于北原沿海内陆首都
