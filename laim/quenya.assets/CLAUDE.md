# CLAUDE.md — Quenya Course Assets

## Project Overview
This folder contains assets for generating Quenya (Elvish language) course exercises, including example sentences, vocabulary, and grammar-based drills in a Duolingo-like format.

## Authoritative Sources
- **灰机wiki**: https://lotr.huijiwiki.com/wiki/昆雅语详细语法
- **Eldamo**: https://eldamo.org
- **NO OTHER SOURCES** — English Wikipedia, Tolkien Gateway, and other sites contain outdated or incorrect information.

## Key Files
- `quenya-grammar-reference.md` — Complete grammar reference (noun declension, verb conjugation, pronouns, syntax)
- `quenya-vocabulary.md` — Vocabulary list (no personal names or place names)
- `quenya-grammar-check.md` — Grammar validation checklist for sentence checking

## Skills

### /quenya-grammar-check
Description: Validate constructed Quenya sentences against grammar rules. Use this skill periodically to check that generated sentences are grammatically correct.
Steps:
1. Read the grammar reference at `quenya-grammar-reference.md`
2. Read the grammar check rules at `quenya-grammar-check.md`
3. For each sentence provided, verify:
   - Noun declension correctness (right case suffix for the right noun class)
   - Verb conjugation correctness (right tense formation for the verb class)
   - Pronoun agreement (subject suffixes vs independent pronouns used correctly)
   - Adjective-noun number agreement
   - Word order (SVO default, adjective position)
   - Proper use of copula (ná- vs ëa-)
   - Negation form (lá- for general, áva for imperative)
   - Article usage (i for definite, none for indefinite)
4. Report any errors found with explanation and corrected form
