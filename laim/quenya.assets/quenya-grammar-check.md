# 昆雅语语法检查清单
# Quenya Grammar Check Rules

本文档用于验证构造的昆雅语句子是否合乎语法。每条规则附带正误示例。

---

## 1. 名词变格检查

### 1.1 V类名词（元音结尾：a, o, u, i）
- [ ] 复数加 -r：cirya → ciryar ✓ | *ciryai ✗
- [ ] 属格 -o 替换尾 -a：cirya → ciryo ✓ | *ciryao ✗
- [ ] 其他格直接加后缀：ciryanna, ciryallo, ciryassë, ciryanen ✓

### 1.2 E类名词（-ë 结尾）
- [ ] 复数 -ë → -i：lassë → lassi ✓ | *lassër ✗
- [ ] 属格 -ë + -o → -ëo：lassëo ✓
- [ ] 复数属格 -ion：lassion ✓

### 1.3 C类名词（辅音结尾）
- [ ] 复数加 -i：atan → atani ✓
- [ ] 属格加 -o：atan → atano ✓
- [ ] 方位格/工具格需介音：atanessë, atananen ✓ | *atanssë ✗

### 1.4 格的语义正确性
- [ ] 与格 -n 用于间接宾语/受益者
- [ ] 属格 -o 用于来源/部分关系
- [ ] 所有格 -va 用于领属/形容词性
- [ ] 向格 -nna 用于方向/目的地
- [ ] 离格 -llo 用于来源/起点
- [ ] 方位格 -ssë 用于位置
- [ ] 工具格 -nen 用于工具/手段

---

## 2. 动词变位检查

### 2.1 不定过去时
- [ ] 基础动词加 -ë：matë, carë, tulë ✓
- [ ] 加主格后缀时 -ë → -i：matin, carin ✓ | *matën ✗
- [ ] 派生动词保持原形：henta, orta ✓
- [ ] 第三人称后缀 -s 前用 -i-：caris ✓ | *carës ✗

### 2.2 现在时
- [ ] 基础动词延长干元音 + -a：mat- → máta ✓
- [ ] r/l/n/m 结尾双写辅音：car- → carra ✓ | *cára ✗
- [ ] 派生动词延长干元音 + -ëa/-ea：henta- → héntea ✓

### 2.3 过去时
- [ ] 词干末 p,t,k → 鼻音中缀：mante, quente, kampe ✓
- [ ] 词干末 r → 加 -në：car- → carnë ✓
- [ ] 词干末 l → 双写 + -ë：tul- → tullë ✓
- [ ] 词干末 n → 双写 + -ë：ken- → kenne ✓
- [ ] 词干末 m → 加 -në：cam- → camne ✓
- [ ] 派生动词加 -në：hentanë, ortanë ✓

### 2.4 完成体
- [ ] 增强元音前缀：a-(默认), i-(i干元音), u-(o/u干元音)
- [ ] 延长干元音 + -ië：amátië, utúlië, acárië ✓
- [ ] 派生动词尾 -a → -ië：orta- → orortië ✓

### 2.5 将来时
- [ ] 所有动词加 -uva：matuva, caruva, hentauva ✓

---

## 3. 代词检查

### 3.1 主语代词
- [ ] 主语代词必须用后缀形式，不可用独立代词作主语
- [ ] 正确：carin "我做" ✓ | 错误：*ni carë ✗
- [ ] 例外：增势代词可作主语：inyë carë ✓

### 3.2 宾语代词
- [ ] 宾格后缀 -s（他/她/它）, -t（他们）
- [ ] 宾格后缀在主格后缀之后：cennenyes "我看见了他" ✓
- [ ] 有宾格后缀时主格后缀必须用长型：-nyë 非 -n

### 3.3 物主代词
- [ ] 附着于名词：atarinya "我的父亲" ✓
- [ ] C类名词需介音 -e-：atanelya "您的人类" ✓

### 3.4 tye vs. lye
- [ ] tye 用于亲密关系（家人、恋人）
- [ ] lye 用于正式/尊敬场合

---

## 4. 形容词检查

### 4.1 数量一致
- [ ] 修饰复数名词时形容词变复数
- [ ] -a → -ë：vanya vendë → vanyë vendi ✓
- [ ] -ëa → -ië：laurëa → laurië ✓
- [ ] -ë → -i：mornë → morni ✓

### 4.2 位置
- [ ] 默认前置于名词
- [ ] 后置也合法（诗歌/强调）

### 4.3 比较级
- [ ] 使用 lá "比"：calima lá elen "比星星明亮" ✓
- [ ] 最高级：an- + 属格复数：elenion ancalima ✓

---

## 5. 句法检查

### 5.1 语序
- [ ] 默认 SVO
- [ ] 代词主语后缀于动词（非独立前置）

### 5.2 系词
- [ ] ná- 用于表述（X是Y）：i atan ná mára ✓
- [ ] ëa- 用于存在（X存在）：Eru ëa ✓
- [ ] 现在时 ná 可省略：i atan mára ✓
- [ ] ëa 不接表语：*Eru ëa mára ✗

### 5.3 否定
- [ ] 一般否定：lá + 动词
- [ ] 否定命令：áva + 动词
- [ ] 不可用 *ú- 作为动词否定前缀

### 5.4 疑问
- [ ] 特殊疑问：man/mana + 动词
- [ ] 是非疑问：ma + 陈述句

### 5.5 定冠词
- [ ] i 用于定指名词（单复数通用）
- [ ] 部分复数 -li 通常不与 i 连用

### 5.6 关系从句
- [ ] i 引导（不变化，最常用）
- [ ] ya/ye 引导（需变格时）

---

## 6. 常见错误模式

| 错误 | 正确 | 说明 |
|------|------|------|
| *ni carë | carin | 独立代词不作主语 |
| *ciryai | ciryar | V类名词复数用 -r |
| *lassër | lassi | E类名词复数 -ë→-i |
| *cára | carra | r结尾基础动词现在时双写 |
| *ciryao | ciryo | 属格 -o 替换尾 -a |
| *vanya vendi | vanyë vendi | 形容词需复数一致 |
| *Eru ëa mára | Eru ëa / Eru ná mára | ëa不接表语 |
| *matën | matin | 加后缀时 -ë→-i |

---

## 7. 检查流程

对每个待检查的句子，按以下顺序验证：

1. **词汇**：每个词是否为合法昆雅语词汇？
2. **名词格**：名词是否使用了正确的格后缀？后缀是否匹配名词类别？
3. **动词时态**：动词变位是否正确？是否匹配动词类别？
4. **代词**：主语代词是否用后缀？宾语代词位置是否正确？
5. **一致性**：形容词是否与名词数量一致？所有格是否与被修饰词一致？
6. **语序**：是否遵循SVO？形容词位置是否合理？
7. **系词/否定**：ná/ëa是否用对？否定形式是否正确？
8. **冠词**：定冠词i是否使用得当？
