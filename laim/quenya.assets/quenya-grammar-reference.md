# 昆雅语语法参考手册
# Quenya Grammar Reference

> 本文档基于 eldamo.org 和灰机wiki（昆雅语详细语法）两个权威来源编写。
> 所有语法规则基于晚期昆雅语（Late Quenya, 1950-1972）。

---

## 1. 音系与转写 (Phonology & Transcription)

### 元音 (Vowels)
- 短元音: a, e, i, o, u
- 长元音: á, é, í, ó, ú（重音总落在长元音上）
- 双元音: ai, oi, ui, au, eu, iu

### 辅音 (Consonants)
- 塞音: p, t, k (c), qu [kw]
- 鼻音: m, n, ñ [ŋ] (仅出现在ng, nk中)
- 流音: l, r
- 擦音: f, s, h, hw [ʍ], hy [ç]
- 近音: v, y, w

### 重音规则
- 双音节词：重音在第一音节
- 三音节及以上：若倒数第二音节为重音节（含长元音、双元音或以辅音丛结尾），重音落在倒数第二音节；否则落在倒数第三音节

### 词尾限制
- 名词仅能以元音或 l, n, r, s, t 结尾

## 2. 名词系统 (Noun System)

### 2.1 名词分类

| 类别 | 定义 | 示例 |
|------|------|------|
| V类 (Vocalic) | 以元音 a, o, u, (i) 结尾 | cirya "船", ondo "岩石" |
| E类 (E-stem) | 以 e 结尾 | lassë "叶子", hendi "眼睛" |
| C类 (Consonantal) | 以辅音 l, n, r, s, t 结尾 | atan "人类", elen "星星" |

### 2.2 数 (Number)

#### 复数
| 类别 | 规则 | 示例 |
|------|------|------|
| V类 | 加 -r | cirya → ciryar |
| E类 | -ë → -i | lassë → lassi |
| C类 | 加 -i（可能恢复古辅音） | atan → atani, elen → eleni |

特殊：以 -ië, -lë 结尾的名词复数加 -r → -ier, -ler

#### 双数
- -t 后缀：用于V类/E类名词 (ciryat "两艘船", lasset "两片叶子")
- -u 后缀：用于C类名词及自然成对物 (hendu "一双眼睛")
- 尾音节含 t/d 的名词倾向用 -u (aldu "双树")

#### 部分复数
- 后缀 -li，表示"一些，许多"（非全部）
- ciryali "一些船", Eldali "一些精灵"
- 对比：Eldar "精灵族（全体）" vs. Eldali "一些精灵"
- C类名词的部分复数因复杂性在现代昆雅语中已少用

### 2.3 格系统 (Case System)

昆雅语有7个格（加上主格/宾格共9种形式）：

| 格 | 后缀 | 功能 | 示例 |
|----|------|------|------|
| 主格/宾格 | (无) | 主语/宾语 | cirya |
| 与格 (Dative) | -n | 间接宾语，"为了" | ciryan |
| 属格 (Genitive) | -o | "……的"（来源/部分） | ciryo |
| 所有格 (Possessive) | -va/-wa | "……的"（领属/形容词性） | ciryava |
| 向格 (Allative) | -nna | "朝向，到" | ciryanna |
| 离格 (Ablative) | -llo | "从，离开" | ciryallo |
| 方位格 (Locative) | -ssë | "在……中/上" | ciryassë |
| 工具格 (Instrumental) | -nen | "用，以" | ciryanen |

#### cirya "船" 完整变格表

| 格 | 单数 | 复数 | 双数 | 部分复数 |
|----|------|------|------|----------|
| 主格 | cirya | ciryar | ciryat | ciryali |
| 与格 | ciryan | ciryain | ciryant | ciryallin |
| 属格 | ciryo | ciryaron | ciryato | ciryallion |
| 所有格 | ciryava | ciryaiva | ciryatwa | ciryalíva |
| 向格 | ciryanna | ciryannar | ciryanta | ciryallinna(r) |
| 离格 | ciryallo | ciryallon/ciryallor | ciryalto | ciryallillo(n) |
| 方位格 | ciryassë | ciryassen | ciryatsë | ciryallissë(n) |
| 工具格 | ciryanen | ciryainen | ciryantem | ciryalínen |

#### lassë "叶子" (E类) 变格要点
- 复数主格: lassi (非 *lasser)
- 单数属格: lassëo
- 复数属格: lassion

#### atan "人类" (C类) 变格要点
- 复数主格: atani
- 单数属格: atano
- 复数属格: atanion
- 单数方位格: atanessë (加介音 -e-)
- 单数工具格: atananen (加介音 -a-)

### 2.4 属格 vs. 所有格

| 属格 -o | 所有格 -va |
|---------|-----------|
| 来源、部分关系 | 领属、形容词性 |
| róma Oromëo "来自欧洛米的号角" | róma Oroméva "欧洛米的号角（他拥有的）" |
| 可前置或后置 | 通常后置 |
| 不需数量一致 | 需与被修饰词数量一致 |

### 2.5 定冠词
- i（单复数通用）：i atan "那个人", i atani "那些人"
- 无不定冠词

## 3. 代词系统 (Pronoun System)

### 3.1 独立代词（宾格形式）

| 人称 | 单数 | 复数 | 双数 |
|------|------|------|------|
| 第一人称 | ni "我" | me "我们(排斥)" / ve "咱们(包含)" | met "我俩" |
| 第二人称 | tye "你(亲密)" / lye "您(尊敬)" | le "你们" | — |
| 第三人称 | se "他/她" / sa "它" | te "他们" / tai "它们" | tu "他俩" |

注意：独立代词通常只作宾语，不作主语。主语用代词后缀。

### 3.2 主格代词后缀（附着于动词）

| 人称 | 单数 | 复数 | 双数 |
|------|------|------|------|
| 1人称 | -n(yë) "我" | -lmë "我们(排斥)" / -lvë "咱们(包含)" | -mmë / -nquë |
| 2人称 | -tyë "你(亲密)" / -lyë "您(尊敬)" | -ldë "你们" | — |
| 3人称 | -s(së) "他/她/它" | -ntë/-ltë "他们" | — |

示例：
- carin "我做" (car- + -n)
- carityë "你做(亲密)" (car-i-tyë)
- caris "他/她做" (car-i-s)
- carilvë "咱们做" (car-i-lvë)

### 3.3 宾格代词后缀

仅有两个确认的宾格后缀：
- -s "他/她/它"
- -t "他们"

示例：cennenyes "我看见了他" (cenne-nye-s)

### 3.4 物主代词后缀

| 人称 | 后缀 | 示例 |
|------|------|------|
| 1sg | -nya "我的" | atarinya "我的父亲" |
| 2sg.pol | -lya "您的" | atarelya "您的父亲" |
| 2sg.fam | -tya "你的" | ataretya "你的父亲" |
| 3sg | -rya "他/她的" | atarerya "他/她的父亲" |
| 1pl.excl | -lma "我们的" | atarelma "我们的父亲" |
| 1pl.incl | -lva "咱们的" | atarelva "咱们的父亲" |

### 3.5 增势代词（强调形式）
- inyë "我自己", elyë "您自己", inque "咱俩"
- 可作主语：Elyë hiruva "您将找到"

### 3.6 指示代词
- sina "这个"（近处）
- tana "那个"（远处）
- enta "那个"（更远处）

### 3.7 关系代词
- i（不变化，最常用）："the man who...", "the thing that..."
- ya（无灵，可变格）：yassën "在其中"
- ye（有灵，可变格）：yello "从他那里"

### 3.8 疑问代词
- man "谁"
- mana "什么"
- manen "如何"
- massë "哪里"
- manna "到哪儿"
- mallo "从哪儿"

## 4. 动词系统 (Verb System)

### 4.1 动词分类

| 类别 | 特征 | 示例 |
|------|------|------|
| 基础动词 | 辅音词根，不定过去时加 -ë | mat- "吃", car- "做", tul- "来" |
| 派生动词（弱动词） | 以 -a 结尾 | henta- "阅读", orta- "升起" |
| A尾动词 | 词根+a，固有进行意味 | fara- "打猎", hora- "等待" |
| U尾动词 | 词根+u，常有开端意味 | liru- "唱歌", nicu- "冻结" |
| ya-动词 | 含 -ya 后缀 | sirya- "流淌", harya- "拥有" |

### 4.2 五种时态

#### 不定过去时 (Aorist) — 一般性陈述、习惯
| 动词类 | 构造 | 示例 |
|--------|------|------|
| 基础动词 | 词干 + -ë（加后缀时 → -i） | matë "吃", matin "我吃" |
| 派生动词 | 词干本身 | henta "阅读" |
| A尾动词 | 词干本身 | fara "打猎" |
| U尾动词 | 词干本身 | liru "唱歌" |

#### 现在时 (Present/Continuous) — 正在进行
| 动词类 | 构造 | 示例 |
|--------|------|------|
| 基础动词 | 延长干元音 + -a | mat- → máta "正在吃" |
| 派生动词 | 延长干元音 + -a（替换尾a） | henta- → héntëa? 或 héntea |
| A尾动词 | 延长干元音 + -a | fara- → fára "正在打猎" |
| U尾动词 | 延长干元音 + -a | liru- → lírua "正在唱" |

注：基础动词词干末辅音为 r/l/n/m 时，现在时加 -a 前辅音双写：
- car- → carra "正在做"（而非 *cára）

#### 过去时 (Past) — 过去发生的事
| 动词类 | 构造 | 示例 |
|--------|------|------|
| 基础动词(强) | 鼻音中缀/后缀 + -ë | mat- → mantë, car- → carnë, tul- → tullë |
| 派生动词(弱) | 词干 + -në | henta- → hentanë, orta- → ortanë |
| A尾动词 | 词干 + -në | fara- → faranë |
| U尾动词 | 词干 + -në | liru- → lirunë |

基础动词强过去时规则（按词干末辅音）：
- p, t, k → 鼻音中缀: kampe, mante, runke
- r, l → 加 -në: carnë, tullë (l双写)
- m → 加 -në: camne
- n → 双写: kenne, tonne
- v (< 古b) → 元音延长或 -mb-: láve 或 kambe

#### 完成体 (Perfect) — 已完成的动作
| 动词类 | 构造 | 示例 |
|--------|------|------|
| 基础动词 | 增强元音 + 延长干元音 + -ië | mat- → amátië, car- → acárië, tul- → utúlië |
| 派生动词 | 增强元音 + 词干(-a→-ië) | orta- → orortië |
| A尾动词 | 增强元音 + 词干(-a→-ië) | fara- → afárië |

增强元音规则：a/e → a-, i → i-, o/u → u-（前缀）

#### 将来时 (Future) — 将要发生
| 动词类 | 构造 | 示例 |
|--------|------|------|
| 所有动词 | 词干 + -uva | mat- → matuva, henta- → hentauva, car- → caruva |

### 4.3 不定式 (Infinitive)
- 基础动词：与不定过去时同形 matë "to eat"
- 派生动词：与不定过去时同形 henta "to read"
- 特殊不定式（带宾语时）：词干 + -ita → carita "to do it"

### 4.4 动名词 (Gerund)
- 后缀 -ië：matië "吃（这件事）", karië "做（这件事）"
- 派生动词：替换尾 -a → -ië：hentië "阅读（这件事）"
- 可作主语：matië mára ná "吃是好的"
- 可变格但不可加代词后缀

### 4.5 分词 (Participles)

#### 主动分词
- 后缀 -la：matila "正在吃的", carila "正在做的"
- 用法类似形容词：i matila atan "正在吃的人"

#### 被动分词
- 后缀 -ina：matina "被吃的", carina "被做的"
- 基础动词直接加 -ina
- 派生动词替换尾 -a → -ina：ortaina "被升起的"

### 4.6 祈使/命令式 (Imperative)
- á/a + 动词不定过去时：á carë! "做！", á tulë! "来！"
- 否定命令：áva carë! "不要做！"

### 4.7 系词 (Copula)

#### ná- "是"（表述性）
| 时态 | 形式 |
|------|------|
| 不定过去/现在 | ná / na（可省略） |
| 过去 | nánë / né |
| 完成 | anáië |
| 将来 | nauva |

- 现在时可省略：i atan (ná) mára "那人（是）好的"

#### ëa- "存在"（存在性）
| 时态 | 形式 |
|------|------|
| 不定过去/现在 | ëa |
| 过去 | engë |
| 完成 | engië |
| 将来 | ëauva |

- 不接表语：Eru ëa "一如存在"

### 4.8 否定 (Negation)
- lá + 动词：lá carë "不做"
- ui "不是"（存在性否定）
- 否定命令：áva + 动词

## 5. 形容词 (Adjectives)

### 5.1 位置
- 通常置于名词前：vanya vendë "美丽的少女"
- 也可后置（尤其在诗歌中）

### 5.2 复数一致
形容词修饰复数名词时需变为复数：
- 以 -a 结尾 → -ë：vanya → vanyë (vanyë vendi "美丽的少女们")
- 以 -ëa 结尾 → -ië：laurëa → laurië
- 以 -ë 结尾 → -i：mornë → morni
- 以辅音结尾 → 不变

### 5.3 比较级与最高级

#### 比较级
- 使用 lá "超过，比"：X calima lá Y "X比Y明亮"
- 或：X ná calima lá Y

#### 最高级
- 前缀 an- + 属格复数：elenion ancalima "星星中最明亮的"
- 前缀 ari-：aricalima "最明亮的"

## 6. 副词 (Adverbs)

### 构造方式
- 形容词 + -vë：mára → máravë "好地"
- 部分副词为独立词：sí "现在", enta "然后", amba "向上"

## 9. 句法 (Syntax)

### 9.1 基本语序
- SVO（主-动-宾）：i atan matë massa "那人吃面包"
- 代词主语后缀于动词：matin massa "我吃面包"

### 9.2 形容词位置
- 通常前置：calima elen "明亮的星"
- 后置也可（诗歌/强调）

### 9.3 "最后可变格词"法则
- 在名词短语中，格后缀附着于短语中最后一个可变格的词上
- ciryar "船（复数）" → ciryar calar "明亮的船们" 但 calar ciryar 也可

### 9.4 疑问句
- 使用疑问代词：man tulë? "谁来了？"
- 是非问句：ma + 陈述句：ma tulis? "他来了吗？"

### 9.5 从属连词
- sa "that"（引导宾语从句）：quetin sa tules "我说他来了"
- i / ya（引导关系从句）：i atan i cennenyë "我看见的那个人"

### 9.6 否定句
- lá + 动词：lá carin "我不做"
- 否定命令：áva + 动词：áva carë! "不要做！"

## 10. 常用句型模板

### 简单陈述
```
[主语] [动词-主格后缀] [宾语]
matin massa. "我吃面包。"
i nér quetë parma. "那男人说了一本书（的事）。"
```

### 系词句
```
[主语] (ná) [表语]
i vendë (ná) vanya. "那少女（是）美丽的。"
```

### 存在句
```
[主语] ëa [方位格]
Eru ëa. "一如存在。"
i elen ëa menelessë. "那颗星在天空中。"
```

### 过去时叙述
```
[动词过去时-主格后缀] [宾语]
cennenyes. "我看见了他。"
i nér mantë massa. "那人吃了面包。"
```

### 将来时
```
[动词将来时-主格后缀] [宾语]
matuvan massa. "我将吃面包。"
hiruvalyë. "您将找到。"
```

### 命令句
```
á [动词]!
á tulë! "来！"
áva quetë! "不要说！"
```

### 疑问句
```
man/mana [动词]?
man tulë? "谁来了？"
ma tulis? "他来了吗？"
```

## 附录A：韵律性延长 (Rhythmic Lengthening)

当一个词以三个或更多连续短音节结尾时，倒数第三个音节的元音会被延长：
- ciryalínen (非 *ciryalinen)
- teluméli (非 *telumeli)

## 附录B：从属连词 sa 与 ha

- sa：引导宾语从句 "that..."：merin sa haryalyë alassë "我希望您拥有快乐"
- ha：引导主语从句（较古老，现代昆雅语中 sa 可兼任）

## 附录C：不定代词

- mo "某人"（作主语）：mo quetë "某人说" → "据说"
- ma "某物"（作宾语）
